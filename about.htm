About This Project

This web application demonstrates a minimalist integration of Stable Diffusion image generation into a static HTML page, using the Pollinations.ai public API.

  • Simplicity: Only plain HTML and vanilla JavaScript are used—no frameworks, no back-end server.
  • Accessibility: No API keys or authentication steps are required. Anyone with a modern browser and internet access can generate images by typing a prompt.
  • Transparency: All source code is released under the MIT License, and the external Pollinations.ai API is acknowledged without bundling or redistributing its code.
  • Educational Value: By stripping away all non-essential layers, this demo helps learners understand how client-side fetch calls can drive real-time AI services.

Whether you’re exploring generative AI for the first time or building a quick prototype for class, this project shows how a few lines of code can unlock powerful image synthesis—all in a single HTML file.
